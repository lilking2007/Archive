## 💡 Key Concepts

### What is Ollama?
A tool for running AI models locally on your computer

### What is Claude Code?
Anthropic's AI coding assistant that you can run on your machine

### What is Qwen?
A powerful open-source AI model excellent at coding tasks

### Why local instead of cloud?
- No API costs
- No token limits
- Works offline
- Your data stays private
- Full control

### What does "localhost:11434" mean?
YOUR computer, not the cloud. That's why it's so fast and private!
